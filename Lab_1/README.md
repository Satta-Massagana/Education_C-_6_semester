# Многопоточная обработка данных

## Описание
Реализация сравнения последовательной и параллельной обработки 10M элементов с использованием базового Thread API.

## Результаты
- Процессор: 4 ядра, без HT
- Размер данных: 10 000 000 элементов  

- Последовательная обработка: 1659 мс
- Параллельная обработка (4 потоков): 421 мс
- Ускорение: 3,94x (лучшее, в среднем 3.8)  

- Последовательная обработка: 1595 мс
- Параллельная обработка (2 потоков): 784 мс
- Ускорение: 2,03x (в среднем 2)  

- Последовательная обработка: 1596 мс
- Параллельная обработка (3 потоков): 516 мс
- Ускорение: 3,09x (лучшее, в среднем 3)  

- Последовательная обработка: 1585 мс
- Параллельная обработка (8 потоков): 464 мс
- Ускорение: 3,42x (лучшее, в среднем 3.2)  

- Последовательная обработка: 1586 мс
- Параллельная обработка (500 потоков): 427 мс
- Ускорение: 3,71x (лучшее, в среднем 3.5)
